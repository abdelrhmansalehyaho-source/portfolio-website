#  Portfolio Website

##  Project Overview

This project is a personal portfolio website developed by **Abdelrahman Saleh** to showcase skills, projects, and experience as a Front-End Developer.
The website is fully responsive and designed with a modern and clean user interface.

---

##  Project Goal

The goal of this project is to build a professional portfolio website that demonstrates front-end development skills and the ability to design and deploy a real-world website.

* HTML5

---

##  Key Features

* Fully responsive design (Desktop, Tablet, Mobile)
* Responsive navigation bar
* Mobile menu toggle
* Smooth scrolling
* Interactive UI effects (hover & animations)
* Contact form with validation
* Password show/hide feature
* Password strength indicator
* Success popup message
* Back-to-top button
